This is a gallery of some results in multi-label image classification I achieved last December. I used an Inception-v4 based convnet architecture which I trained for 2-3 hours on a dataset of more than 50,000 200x200px images scraped from unsplash.com. Predicted labels at the top are in arbitrary order. Only labels above a certain cutoff confidence value are shown.
