mnist_captcha Dataset

June 15, 2018 ⧗ 1 minute read
→ Preview
→ Download

This dataset consists of about 270,000 MNIST-like captcha images. Each 84x30px image consists of 5 alphanumeric characters. I found the dataset useful for testing the UberAI Labs CoordConv method[1]. For interesting results you shouldn't use all 270k images. Also remember to split the dataset into a train and test set.

The filenames are structured like this: 
captcha content__font.png
